# 🎓 Kai Micro-Gig & Peer Help Marketplace

> **Pocket Money, Real Skills, Trusted Inside Your College**

A campus-based web platform that connects students who need help with students who have skills. This project creates a micro-gig economy inside college ecosystems where students can post small tasks, apply for gigs, earn rewards, and build real project experience.

## 📌 Problem Statement

Students want:
- Pocket money opportunities  
- Skill-based real work experience  
- Portfolio projects  
- Trusted peer collaboration  

However, there is no structured marketplace inside colleges to connect students who need help with those who have skills.

---

## 💡 Solution

Kai Micro-Gig Marketplace provides a platform where students can:

- Post small tasks (gigs)
- Apply for available gigs
- Collaborate with peers
- Track applications
- Build proof-of-work portfolios

This creates a **trusted campus skill economy**.

---

## ✨ Key Features

- 🔐 User Signup & Login
- 📝 Post Gig (request work from peers)
- 🔎 View Available Gigs
- 📩 Apply for Gigs
- 📊 Track Applications Status
- 👤 User Profile Management
- 🤖 Chatbot Assistance (UI feature)
